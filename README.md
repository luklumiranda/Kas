## Aksi & Fitur Utama
Aplikasi ini memungkinkan pengguna untuk mengelola keuangan dengan kontrol penuh melalui aksi-aksi berikut:

Manajemen Akun: Pengguna dapat mendaftar, login, dan mengelola profil keamanan.

Pencatatan Transaksi: Menambahkan data pemasukan atau pengeluaran kas secara mendetail.

Audit Riwayat: Melihat log transaksi yang sudah dilakukan untuk keperluan monitoring.

Visualisasi Saldo: Menampilkan total saldo akhir secara otomatis berdasarkan kalkulasi database.
